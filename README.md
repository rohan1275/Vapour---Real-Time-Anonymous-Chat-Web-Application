Vapour is a privacy-focused real-time anonymous chat platform that allows users to create temporary private rooms with invite-based access. Conversations are ephemeral and automatically deleted when the session ends.

## Features
- Private room-based anonymous chat
- Real-time messaging using Firebase Firestore
- Typing indicators
- Ephemeral room lifecycle and message deletion
- Responsive modern UI with animations
- Cloud-integrated architecture

## Tech Stack
- React.js
- Firebase Firestore
- JavaScript
- HTML/CSS
- Framer Motion

## Purpose
This project demonstrates real-time frontend systems, cloud database integration, and modern UI engineering for frontend and cloud roles.
